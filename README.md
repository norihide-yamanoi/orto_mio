・rails 6.0.5
# 就業Termの技術
・お気に入り機能

# カリキュラム外の技術
・geocoder  
・ransack
# 実行手順  
$ git clone github.com/norihide-yamanoi/orto_mio  
$ cd orto_mio  
$ bundle install  
$ rails db:create   
$ rails db:migrate  
$ rails s
